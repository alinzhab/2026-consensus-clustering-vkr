Среднее значение метрики по алгоритмам

Сравнение четырёх алгоритмов консенсус кластеризации. Фильтры применяются к набору датасетов и правилам linkage.

Тепловая карта: алгоритм × датасет

Каждая ячейка — среднее значение выбранной метрики. Позволяет увидеть, на каких датасетах и каким алгоритмом достигается лучший результат.

QD-selection vs Random: ΔNMI по датасетам

Разность NMI(QD) − NMI(random) для каждого алгоритма и датасета. Положительные значения: QD-selection улучшает результат. Данные берутся из файла results/qd_selection_analysis.tsv, если он существует.

Нет данных QD-эксперимента. Запустите: python experiments/run_qd_selection_analysis.py

Среднее качество по типам данных

Позволяет ответить на вопрос: на каких типах данных (биологические, синтетические вытянутые, высокоразмерные и т.д.) каждый алгоритм работает лучше или хуже. Это ключевой анализ для формулировки выводов ВКР.

Время работы алгоритмов (QD-эксперимент)

Runtime в секундах для каждого алгоритма и датасета из QD-эксперимента. Иллюстрирует вычислительную стоимость SDGCA относительно иерархических методов.

Нет данных QD-эксперимента.
Ablation Study

SDGCA Modified: декомпозиция вклада компонентов

Сравнение шести вариантов: A (baseline), B (modified, fixed t), C (modified, adaptive t), и их QD-версий (D/E/F). Нечёткие веса, heat-kernel диффузия и adaptive diffusion time изолированы от эффекта QD-selection. Для запуска: python experiments/run_sdgca_modified_ablation.py --profile mini --m 8 --runs 3

NMI по вариантам (среднее по датасетам)

Каждый столбец — среднее NMI варианта на всех датасетах. A = baseline, B = modified fixed t=1.0, C = modified adaptive t. D/E/F — те же варианты с QD-selection.

ΔNMI: adaptive t − fixed t (C − B) по датасетам

Положительное значение → adaptive diffusion time улучшает результат по сравнению с фиксированным t=1.0. Нейтральные значения ожидаемы на датасетах с NMI≈1.0 базово.

Runtime: сравнение вариантов (секунды)

Иллюстрирует накладные расходы модификаций. Если C (adaptive) близко к B (fixed) — это сильный аргумент: adaptive t добавляет negligible overhead.

Тепловая карта ablation: вариант × тип данных

NMI в разрезе типов данных. Позволяет увидеть, где modified помогает, а где модификация нейтральна или хуже.

Эффект QD-selection на SDGCA modified (E−B, F−C)

Изолированный эффект QD-selection на modified variants. E−B: QD vs random при fixed t. F−C: QD vs random при adaptive t.

Runtime vs n (число объектов)

Точки: один датасет = одно n. Цвет = вариант ablation. По оси Y — wall-clock за полный вызов алгоритма (все прогоны runs). Для оценки масштаба см. также секцию «Runtime scaling» в sdgca_modified_ablation_summary.md (подгонка time ≈ a · nb).

Adaptive diffusion time: распределение по датасетам

Значения t, вычисленные через спектральный зазор. Показывает, как сильно adaptive t отклоняется от t=1.0 на разных типах данных.

Ablation-данные отсутствуют.
